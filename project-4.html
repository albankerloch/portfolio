Pong en ligne

Création d'un site web permettant aux élèves de l'école 42 de jouer à Pong en ligne

Github

Le projet

Le site répond à plusieurs standards du web moderne : il est responsive pour pouvoir jouer sur tous les OS, sa base de données est cryptée pour sécuriser les données des utilisateurs et son contenu est fortement dynamique.

Les fonctionnalités :
- Se connecter avec un compte de l'école 42 (sécurisation via OAuth 2.0)
- Jouer une partie de Pong avec une autre personne connectée
- Affichage des scores des meilleurs joueurs
- Discuter avec d'autres élèves via un chat (via des websockets)
- Modération possible par les comptes administrateurs du site
