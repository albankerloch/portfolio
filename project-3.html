Un Bash Basique

Création d'un programme fonctionnant comme Bash sous Mac

Github

Le projet

Le programme permet de lancer des programmes dans un environnement Unix.

Les fonctionnalités :
- Afficher un prompt à l'utilisateur
- Trouver et lancer l'éxécutable adéquat (via le PATH ou un chemin absolu ou relatif)
- Prendre en compte les redirections >, <, >>, <<
- Quitter sans fuite mémoire après les commandes CTRL+D, CTRL+C et CTRL+\
- Prendre en compte les pipes |
- Gérer les variables d'environnement et la commande $?
- Prendre en compte les simples et doubles guillemets : ' et "
- Implémenter les fonctions echo, cd, pwd, export, unset, env, exit

Technologies utilisées

C++
HTTP
